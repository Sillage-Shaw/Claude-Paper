使用 `session-note` skill 将用户指定的 AI 交互过程、排错过程、环境配置过程或工具安装过程整理成可复用 Markdown 笔记。

用户输入如下：

$ARGUMENTS

执行要求：

1. 严格按照 `.claude/skills/session-note/SKILL.md` 的流程执行；
2. 所有文件路径、文件创建规则、追加规则和最终回复格式都遵循根目录 `CLAUDE.md`；
3. 如果用户给出了明确路径、标题、范围或特殊要求，优先遵循用户要求；
4. 不要记录完整聊天流水账，只提炼可复用信息；
5. 重点记录问题现象、报错信息、环境信息、排查过程、根因、最终解决方案和验证方式；
6. 删除或脱敏 API key、token、密码、私钥等敏感信息；
7. 如果上下文不完整，请说明缺失信息，并基于已有信息整理。
